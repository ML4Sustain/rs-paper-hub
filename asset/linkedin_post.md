## English Version

🛰️ Excited to share **RS Paper Hub** — an open-source tool for the remote sensing research community!

We built a fully automated pipeline that scrapes, cleans, classifies, and visualizes **5,300+ remote sensing papers** from arXiv, making it easy to stay on top of the latest research.

### What it does:

📄 **Automated scraping** — Fetches papers from arXiv API with incremental updates
🧹 **Smart cleaning** — Extracts code repo URLs from abstracts, fills metadata gaps
🔍 **VLM filtering** — Identifies 920+ Vision-Language Model papers using 150+ keyword patterns
🏷️ **Auto-classification** — Labels papers as Method / Dataset / Survey / Application
📊 **Interactive web viewer** — Search, multi-dimensional filtering, chart-based exploration, BibTeX export

### Key highlights:

- One-click pipeline: `python pipeline.py` does everything
- Daily workflow friendly: `python main.py --update` grabs the latest papers
- Beautiful web UI with chart-driven filtering — click a year or category to instantly narrow results
- 100% open source under MIT license

### Built for researchers who want to:
✅ Track the latest remote sensing papers effortlessly
✅ Explore trends across years, topics, and categories
✅ Find VLM-related remote sensing work quickly
✅ Export BibTeX citations in one click

## 中文版本

🛰️ 很高兴分享 **RS Paper Hub** — 一个面向遥感研究社区的开源工具！

我们构建了一套全自动化流程，能够从 arXiv 抓取、清洗、分类和可视化 **5300+ 篇遥感论文**，帮助研究人员轻松掌握最新研究动态。

### 核心功能：

📄 **自动抓取** — 通过 arXiv API 获取论文，支持增量更新
🧹 **智能清洗** — 自动从摘要中提取代码仓库链接，补全元数据
🔍 **VLM 筛选** — 基于 150+ 关键词模式识别 920+ 篇视觉语言模型相关论文
🏷️ **自动分类** — 将论文标注为 Method / Dataset / Survey / Application
📊 **交互式网页浏览器** — 搜索、多维度筛选、图表驱动探索、BibTeX 一键导出

### 亮点：

- 一键流水线：`python pipeline.py` 完成所有步骤
- 日常更新友好：`python main.py --update` 只抓取最新论文
- 图表可点击筛选 — 点击年份或分类即可快速定位
- MIT 开源协议，完全免费

### 适合需要以下场景的研究人员：
✅ 轻松追踪遥感领域最新论文
