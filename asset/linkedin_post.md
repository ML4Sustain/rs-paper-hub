## English Version

🛰️ Excited to share **RS Paper Hub** (https://rspaper.top) — an open-source platform that helps the remote sensing & earth observation community stay on top of the latest research!

We built a fully automated pipeline that scrapes, cleans, classifies, and visualizes **5,600+ remote sensing and earth observation papers** from arXiv — updated daily, zero manual effort.

### What it does:

📄 **Daily automated updates** — Fetches the latest papers from arXiv every day, covering both "remote sensing" and "earth observation"
🏷️ **Multi-dimensional tagging** — Auto-classifies papers by category (Method / Dataset / Survey / Application) and by task (Classification, Object Detection, Change Detection, Segmentation, VQA, Captioning, and more — 11 task types in total)
🔍 **VLM tracking** — Identifies 1,000+ Vision-Language Model papers using 150+ keyword patterns
📊 **Interactive web explorer** — Search, filter by year/type/category/task, chart-driven exploration, and one-click BibTeX export
🔗 **Code discovery** — Automatically extracts code repository links from abstracts

### Highlights:

- **Daily CI/CD pipeline** — Automatically runs every day, so the website is always up to date
- **Incremental processing** — Only new papers go through the pipeline; existing data is preserved
- **Click-to-filter charts** — Click any bar in the year/type/task charts to instantly narrow results
- **Quick date filters** — "Today" and "This Week" buttons to jump straight to the freshest papers
- 100% open source under MIT license

### Built for researchers who want to:

✅ Track the latest remote sensing & earth observation papers effortlessly
✅ Discover research trends across years, tasks, and categories
✅ Find VLM-related remote sensing work at a glance
✅ Export BibTeX citations in one click

Feedback, suggestions, and contributions are very welcome — feel free to open an issue or reach out!
